# PM Methodology Patterns

Use this reference for product management method videos.

## Default Structure

```text
Hook -> Context -> Framework -> Example -> Pitfall -> Recap
```

## Common Visual Templates

| Topic | Best Visual |
|---|---|
| MVP | false definition vs true definition, assumption test card |
| PRD | document anatomy, decision map, missing-context warning |
| User story | actor-goal-benefit card, bad vs good rewrite |
| Prioritization | value-effort matrix, RICE table, tradeoff slider |
| North star metric | metric tree, leading vs lagging indicators |
| Funnel | conversion funnel, drop-off callouts |
| Competitive analysis | comparison matrix, positioning map |
| Roadmap | now-next-later timeline |

## Script Template

```text
Hook:
  One sentence that corrects a common misunderstanding.

Context:
  Why this mistake happens in real product work.

Framework:
  3 to 5 steps. Each step should be a verb phrase.

Example:
  A small concrete product scenario.

Pitfall:
  One thing people misuse or over-apply.

Recap:
  One sentence the viewer can remember.
```

Use plain workplace language. Avoid MBA jargon unless the term itself is the topic.
